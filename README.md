# LLM Resume Analyzer & Career Recommendation System

This project is a Resume Analyzer that uses Large Language Models (LLMs) and Machine Learning techniques to analyze resumes and recommend suitable career paths.

## Features
- Resume parsing from PDF
- Skill extraction using NLP
- Resume analysis using LLM
- Job role recommendations
- Career guidance insights

## How It Works
1. Resume is uploaded by user
2. Text is extracted using NLP
3. Skills are identified
4. LLM analyzes strengths & weaknesses
5. Job roles are recommended based on skill match

## Technologies Used
- Python
- Streamlit
- NLP
- LLM (OpenAI)
- Machine Learning concepts
